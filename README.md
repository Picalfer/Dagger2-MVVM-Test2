# Dagger2 MVVM Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Dagger2](https://img.shields.io/badge/Dagger-2-red.svg)](https://dagger.dev/dev-guide/)
[![Architecture](https://img.shields.io/badge/Pattern-MVVM-blueviolet.svg)](https://developer.android.com/topic/architecture)

Тестовое приложение для изучения Dagger 2 в сочетании с MVVM архитектурой. Демонстрирует внедрение зависимостей в ViewModel и работу с LiveData.

## Функциональность

- Внедрение ViewModel через Dagger 2
- Работа с LiveData и наблюдение за изменениями
- CRUD операции с данными
- Singleton scope для базы данных и ViewModel

## Технологии

- Kotlin
- Dagger 2
- MVVM Architecture
- LiveData
- ViewModel
